# ملاذ للتسويق 🎯

منصة تسويق ذكية لـ iHerb وAmazon مع مزامنة GitHub السحابية

## هيكل الملفات

```
malaz/
├── index.html          ← الصفحة الرئيسية (لا تعدّل إلا النصوص)
├── css/
│   └── style.css       ← التصميم والألوان (عدّله بحرية)
├── js/
│   ├── config.js       ← ★ الإعدادات الافتراضية (هاشتاقات، فئات، منصات)
│   ├── storage.js      ← المزامنة مع GitHub
│   ├── ui.js           ← واجهة المستخدم
│   ├── settings.js     ← إدارة الإعدادات
│   ├── generate.js     ← توليد المحتوى
│   ├── products.js     ← إدارة المنتجات
│   ├── queue.js        ← قائمة الانتظار والنشر
│   └── app.js          ← نقطة البداية
└── README.md
```

## الملف الذي تريد تعديله

| ما تريد تغييره | الملف |
|---|---|
| الألوان والتصميم | `css/style.css` |
| الهاشتاقات الافتراضية | `js/config.js` |
| الفئات الافتراضية | `js/config.js` |
| المنصات الافتراضية | `js/config.js` |
| نموذج AI للتوليد | `js/config.js` |
| نصائح التسويق | `js/ui.js` |

## رفع على GitHub Pages

1. أنشئ Repository جديد على GitHub
2. ارفع كل الملفات مع الحفاظ على نفس هيكل المجلدات
3. Settings → Pages → Branch: main → Save
4. انتظر دقيقة، رابطك: `https://اسمك.github.io/اسم-الريبو`

## الاستخدام

1. افتح الرابط في أي متصفح
2. أدخل:
   - مفتاح Anthropic API من `console.anthropic.com`
   - مفتاح GitHub Token من `github.com/settings/tokens` (Gist scope فقط)
3. ابدأ الاستخدام — كل البيانات تُحفظ تلقائياً

## المزامنة

- كل إضافة أو تعديل يُحفظ في GitHub Gist الخاص بك خلال 3 ثواني
- افتح من أي جهاز وكل بياناتك موجودة
- تكلفة التغريدة الواحدة: ~$0.003
